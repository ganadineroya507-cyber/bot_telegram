# 🚀 BOT TELEGRAM - SISTEMA DE MONETIZACIÓN

Sistema automatizado de bot de Telegram para generar ingresos mediante:
- 📺 Visualización de anuncios (Adsterra + Monetag)
- 📋 Tareas pagadas
- 👥 Sistema de referidos
- 🏆 Ranking de usuarios
- 🎁 Bono diario
- 💸 Solicitudes de retiro manual

---

## ⚙️ Características

- ✔️ Registro único por usuario
- ✔️ Sistema antifraude básico
- ✔️ Límite de anuncios diarios
- ✔️ Temporizador obligatorio (15s - 30s)
- ✔️ Pagos por tareas y anuncios
- ✔️ Panel web administrativo
- ✔️ Base de datos SQLite
- ✔️ Seguimiento de referidos

---

## 💰 Monetización

- Anuncios: $0.000125 por visualización  
- Tareas: $0.0035 por tarea  
- Bono diario: $0.03  
- Límite: 100 anuncios por día  

---

## 💳 Métodos de retiro

## 🛠️ Tecnologías

## 🔒 Seguridad
